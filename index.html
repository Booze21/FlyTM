This page needs a local server

You opened this file directly (file://), but some browser features work best when served over http://. Everything works fine over a local server.

Run a tiny local server from this folder:

# Python 3 (built in on Mac/Linux)
python -m http.server 8000
# Node.js
npx serve .

Then open http://localhost:8000 in your browser.
